\section{Specific Objectives}
 \begin{enumerate}

     Build a scalable UWB-based localization subsystem (anchors + tags) that provides continuous position updates for thousands of devices

     \item  \textbf{Real-time point-cloud visualizer}
     
     Visualize the live 2D point cloud of localized devices overlaid on an arena map (stadium/venue seating plan) shown in Figure \ref{fig:pointcloud_ui}, Where the map of the arena can be configured using exact dimensions.
